USO DEL SCRAPER FACEBOOK MARKETPLACE

1. Coloca tu archivo cookies.json en la raíz del proyecto.
2. Edita fb_marketplace_scraper.py y pon tu URL de perfil de Marketplace en PROFILE_URL.
3. Ejecuta:
   python fb_marketplace_scraper.py
4. El script detecta sesión, scrapea productos y muestra el resultado.

No necesitas otros scripts auxiliares. Todo el flujo está integrado y limpio.
